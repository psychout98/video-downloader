  Media Downloader — Siri Shortcut & Network Setup Guide
══════════════════════════════════════════════════════════════

STEP 1 — NETWORK ACCESS (Tailscale — Recommended)
───────────────────────────────────────────────────
Tailscale gives your phone a private VPN tunnel to your HTPC
so the Siri Shortcut can reach it from anywhere.

1. On your HTPC (Windows):
   • Download Tailscale: https://tailscale.com/download/windows
   • Install and sign in with a free account.
   • Your HTPC will get a stable Tailscale IP like 100.x.x.x
   • Note that IP — you'll need it for the Shortcut.

2. On your iPhone:
   • Install Tailscale from the App Store.
   • Sign in with the SAME Tailscale account.
   • Enable the VPN in Tailscale.

3. Test the connection:
   • Open Safari on your iPhone with Tailscale active.
   • Go to: http://100.x.x.x:8000  (use your HTPC's Tailscale IP)
   • You should see the Media Downloader web UI.


STEP 2 — CREATE THE SIRI SHORTCUT
────────────────────────────────────
On your iPhone, open the Shortcuts app and create a new Shortcut:

   Action 1:  "Ask for Input"
              Prompt:  "What would you like to download?"
              Input type: Text
              (Save result as: "DownloadQuery")

   Action 2:  "Get Contents of URL"
              URL:     http://100.x.x.x:8000/api/download
                       ↑ replace with your HTPC's Tailscale IP
              Method:  POST
              Headers:
                  Content-Type:   application/json
                  Authorization:  Bearer YOUR_SECRET_KEY_HERE
                                  (must match SECRET_KEY in .env)
              Request Body: JSON
                  {
                    "query": [DownloadQuery variable]
                  }
              (Save result as: "APIResponse")

   Action 3:  "Get Value for Key" (or "Dictionary")
              Dictionary: APIResponse
              Key: message
              (Save result as: "ResponseMessage")

   Action 4:  "Show Notification"
              Title:  Media Downloader
              Body:   [ResponseMessage variable]

Name the Shortcut: "Download Movie" (or whatever you like)

Now say "Hey Siri, Download Movie" and it will prompt you for
the title, then send it to your HTPC.


STEP 3 — IMDB SHARE SHEET (Alternative / Bonus)
────────────────────────────────────────────────
You can also share directly from the IMDb app:

1. Create another Shortcut with:
   Action 1: Receive "URL" as input from "Share Sheet"
             (Check "No Input" → "Continue")
   Action 2: "Get Contents of URL"
              URL: http://100.x.x.x:8000/api/download
              Method: POST
              Headers: Content-Type + Authorization (same as above)
              Body: { "query": [Shortcut Input] }
   Action 3: "Show Notification" with the response message

2. In IMDb app:
   • Find a movie/show → tap Share → select your Shortcut
   • The IMDb URL is sent directly as the query — the server
     extracts the IMDb ID automatically.


QUERY SYNTAX REFERENCE
───────────────────────
The server parses natural language queries:

  "Inception"              → search for movie
  "Breaking Bad"           → search for TV show (all seasons)
  "Breaking Bad S01"       → TV show, Season 1
  "Breaking Bad S01E03"    → TV show, S01E03 specifically
  "Breaking Bad season 2"  → TV show, Season 2
  "Attack on Titan"        → detected as anime (via TMDB genres)
  "https://imdb.com/title/tt0903747/"  → direct IMDb lookup

QUALITY SELECTION
────────────────────
The server automatically selects the best available quality:
  Resolution:  4K/2160p > 1080p > 720p
  Source:      Remux > BluRay > WEB-DL > WEBRip
  HDR:         Dolby Vision > HDR10+ > HDR10 > SDR
  Audio:       Atmos > TrueHD > DTS:X > DTS-HD > DD+
  Channels:    7.1 > 5.1 > stereo
  Codec:       HEVC/x265 preferred

Real-Debrid cached torrents are always prioritised (instant
availability). Non-cached torrents will be queued to RD and
downloaded once cached (may take 5–60 minutes for new releases).
